# Issue Tracker & Repository Management

A web-based application to manage repositories and track issues efficiently. Users can create repositories, add issues, update issue details, and view all issues related to a repository. Built with **React** for frontend and **Node.js / Express** with **MongoDB** for backend.

---

## Features

- **Repository Management**
  - Create new repositories
  - Edit repository details
  - View all repositories for a user

- **Issue Tracking**
  - Add new issues to a repository
  - Edit or delete issues
  - View all issues for a repository
  - Track issue status (`open` / `closed`)

- **UI / UX**
  - Clean card-based design
  - Responsive and interactive interface
  - Loading states and informative messages

- **Tech Stack**
  - **Frontend:** React, React Router, Tailwind CSS
  - **Backend:** Node.js, Express
  - **Database:** MongoDB
  - **Other:** Axios / Fetch for API calls
